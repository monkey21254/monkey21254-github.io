그래프 여러개 보여주기

  • subplot
  • plt.subplot(n1, n2, n)
  • n1 : 전체 그림의 세로 개수
  • n2 : 전체 그림의 가로 개수
  • n : 현재 그림의 위치(좌상단부터 1,2,3,…)
    &nbsp&nbsp: ※ 0이 아니고 1부터 시작

  1. plt.figure

  2. plt.subplots_adjust

  3. plt.subplot

  4. plt.subplots_adjust(wspace, hspace) : wspace 가로 공백, hspace 세로 공백
